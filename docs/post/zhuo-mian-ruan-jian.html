🧹 系统优化与清理

  1. BleachBit (www.bleachbit.org): 一款开源系统清理工具，深度清理隐私缓存和垃圾文件，支持Linux和Windows。
  2. Wise Disk Cleaner (www.wisecleaner.com/wise-disk-cleaner.html): 专注于磁盘清理和碎片整理，界面直观，清理效果好。
  3. Glary Utilities (www.glarysoft.com): 功能全面的系统优化与维护工具箱，包含清理、修复、启动管理等模块。
  4. Ashampoo WinOptimizer (www.ashampoo.com/winoptimizer): 德国公司出品的系统优化套件，功能丰富，设计精良。
  5. Autoruns (docs.microsoft.com/sysinternals): 微软Sysinternals出品的利器，可深度管理系统所有自启动项和计划任务。
  6. Process Explorer (docs.microsoft.com/sysinternals): “任务管理器”的终极增强版，可查看进程详细信息及关联资源。
  7. Process Hacker (processhacker.sourceforge.io): 类似Process Explorer的开源进程查看和管理工具，功能强大。
  8. Spacesniffer (www.uderzo.it/main_products/space_sniffer): 以树状图区块形式直观展示磁盘空间占用的神器。
  9. WizTree (diskanalyzer.com): 最快的磁盘空间分析工具之一，利用NTFS MFT表在几秒内完成全盘扫描。
  10. WinDirStat (windirstat.net): 经典的可视化磁盘空间统计工具，通过树形图和文件类型色块进行分析。
  11. TreeSize Free (www.jam-software.com/treesize_free): 快速扫描并显示文件夹大小的工具，可快速定位大文件。
  12. Everything Toolbar (github.com/stnkl/EverythingToolbar): 将Everything的搜索框直接集成到Windows任务栏，体验更佳。
  13. QuickLook (pooi.moe/QuickLook): 在文件资源管理器中按空格键快速预览文件内容（如图片、文档、代码），类似macOS功能。
  14. EarTrumpet (eartrumpet.app): Windows系统托盘增强型音量控制工具，可单独调节每个应用程序的音量。
  15. ModernFlyouts (github.com/ModernFlyouts-Community/ModernFlyouts): 用现代化的、可自定义的弹窗替换Windows默认的音量、亮度等提示框。
  16. PowerToys (github.com/microsoft/PowerToys): 微软官方出品的系统效率工具集，包含分屏增强、快速启动器、重命名工具等。
  17. AltDrag (stefansundin.github.io/altdrag): 允许你通过按住Alt键并拖动鼠标来移动或调整任何窗口，极大提升窗口操作效率。
  18. LiquidText (www.liquidtext.net): 革命性的PDF和文档阅读/批注工具，支持自由关联想法、创建思维导图式笔记。
  19. Mouse without Borders (www.microsoft.com/mousewithoutborders): 微软车库项目，允许用一套键鼠控制多台电脑，类似软件KVM。
  20. Ditto Clipboard Manager (ditto-cp.sourceforge.io): 强大的剪贴板历史管理工具，支持搜索、分组和永久保存复制内容。

  19. Nimbus Note (nimbusweb.me/note): 功能丰富的笔记应用，支持富文本、任务清单和团队协作。
  20. Simplenote (simplenote.com): Automattic公司出品的极致简约的跨平台同步笔记应用，完全免费。
